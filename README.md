# Lights Out Game

A  logic puzzle game built using Java and the Swing framework. The goal is to turn all the lights off on the grid to win. 

# How to Play
 1. The Challenge: During actual gameplay of the puzzle, clicking a button toggles its state (on/off) along with all its adjacent neighbors (up, down, left, right).
 2. The Goal: Strategically click buttons until every light on the board is dark.
 3. Manual Mode: Includes a specialized manual mode for custom board setups.

# Key Features
 1. Auto-Reset: The game automatically re-randomizes upon completion so you can jump straight into the next round.
 2. Win Tracker: Built-in win counter that tracks your victories in regular gameplay mode.

# Getting Started To Play the Game
 1. Clone the repo.
 2. Open the project in your IDE.
 3. Run `LightsOutProgram.java` to start the game.
